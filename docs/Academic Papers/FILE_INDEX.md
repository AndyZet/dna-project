## Naming Convention

**Format:** `YYYY_AuthorLastName_Title.pdf`

- **Year first** (4 digits)
- **Author last name** (or first author + "etal" for multiple authors)
- **Abbreviated title** (key words, ~30-40 characters)
- **File extension**

**Special cases:**
- Alternative versions: `_alt` suffix
- Multiple authors: `_etal` suffix
- Non-English titles: Preserved in original language with English translation in filename
