# Academic Papers Organization Plan

## Proposed Naming Convention

Format: `YYYY_AuthorLastName_Title.pdf`

Examples:
- `2023_Stolarek_Genetic_history_East_Central_Europe.pdf`
- `2019_Margaryan_Population_genomics_Viking_world.pdf`
- `2013_Buko_etal_Bodzia_cemetery.pdf`

## Proposed Directory Structure
