# Academic Papers - Organization Guide

## Naming Convention

Files are organized using the following naming convention:
**`YYYY_AuthorLastName_Title.pdf`**

For multiple authors: **`YYYY_FirstAuthorLastName_et_al_Title.pdf`**

For files without clear author: **`YYYY_Topic_Description.pdf`**

## Directory Structure

### 01_Genetic_Studies/
- Population genetics, aDNA, Y-DNA, mtDNA studies
- Focus: Genetic heritage, population genomics, archaeogenetics

### 02_Medieval_Rus_Rurikids/
- Studies on Kievan Rus', Rurikids, Novgorod
- Focus: Political organization, ethnogenesis, princely families

### 03_Bodzia_Cieple_Archaeology/
- Archaeological studies on Bodzia cemetery and Ciepłe site
- Focus: Elite burials, medieval archaeology, Piast period

### 04_Historical_Studies/
- Historical and political studies
- Focus: Medieval history, naming practices, political organization

### 05_Reference_Materials/
- HTML files, CSV data, reference documents
- Focus: Supporting materials, data extracts, reference sources

## File Organization Rules

1. **Remove duplicates**: Files with "(1)" suffix are duplicates - keep the one with better quality/complete title
2. **Standardize dates**: All dates in brackets [YYYY] format
3. **Author names**: Use last name only, first initial if needed for disambiguation
4. **Titles**: Truncate long titles to ~50 characters, preserve key words
5. **Special characters**: Replace with underscores or remove
